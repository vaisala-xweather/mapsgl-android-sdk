Class TileLayer

  • All Implemented Interfaces:
    com.xweather.mapsglmaps.events.EventSource , com.xweather.mapsglmaps.layers.tile.MapLayer , com.xweather.mapsglmaps.map.MapEventObserver 
    
public class TileLayer
 implements MapLayer<TileData, LayerRenderContext<Any>>

    Represents a layer that renders data from a tile-based data source. This is the base class for all tile-based layers and is not intended to be used directly.
