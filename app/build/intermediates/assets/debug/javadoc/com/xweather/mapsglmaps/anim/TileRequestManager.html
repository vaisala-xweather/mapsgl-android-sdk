Class TileRequestManager

  • All Implemented Interfaces:
    
public final class TileRequestManager<Data extends Object>

    This class is responsible for keeping track of the time series intervals and which time intervals we have data for for each tile. This information is then used to determine which intervals we can actually show during an animation, which should only happen if we have data for all visible tiles for the requested time interval.
