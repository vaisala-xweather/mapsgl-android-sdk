Package com.xweather.mapsgl.coordinates

Class MapBounds

  • All Implemented Interfaces:
    
public final class MapBounds<Space extends MapCoordinateSpace>

    A 2D coordinate on a map or the world, in one of many CoordinateSpaces or zoom levels of some. Not intended to represent a zoomLevel as a 3rd dimension— this is where on the 2D map you are, with any CoordinateSpace zoom levels already figured in. (Any potential 3rd dimension would be altitude.)
